# set-lang

Set-lang is a programming language for computing with hereditarily-finite sets, i.e. sets whose elements are all finite.

In set-lang, the only type is `SET`. That means that all elements within a set are also of type `SET`! This is axiom 1.1 of set-lang.

A complete description of the spec of set-lang can be found in [SPEC.md](./SPEC.md)
